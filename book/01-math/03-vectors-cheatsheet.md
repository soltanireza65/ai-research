# Vectors — Cheatsheet

Quick review of [Vectors](03-vectors.md).

## Core

- Vector = ordered list of numbers; magnitude \(\|v\|\), addition, scalar multiply
- Dot product: \(u \cdot v = \sum u_i v_i\) — alignment (preview for attention)

## Python

```python
import numpy as np
v = np.array([3, 4])
np.linalg.norm(v)  # magnitude
u @ v              # dot product
```
